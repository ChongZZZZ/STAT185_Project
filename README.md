# STAT185_Project

## Setup

1. Create and activate a Python environment
2. Install the required packages:

   ```bash
   pip install -r requirements.txt
   ```


## Running the Examples

In the `VAE/` folder, you will find two example notebooks:

- `example1.ipynb`
- `example2.ipynb`

### Suggested Usage

- **`example1.ipynb`**  
  This notebook is more computationally intensive and is **recommended to be run on a GPU**.
- **`example2.ipynb`**  
  This notebook is lighter and can be run on **CPU** without issues.
